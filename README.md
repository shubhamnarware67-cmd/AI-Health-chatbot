# 🏥 AI Health Assistant
 
> *"Finally, an app that actually listens to your symptoms... unlike some doctors."* 😅
 
A smart, beginner-friendly **ML Health Project** built with **Streamlit** — no hospital visits required.
 
---
 
## 🤔 What Is This?
 
This is an **AI-powered health chatbot** that does 3 things your mom already tries to do — but smarter:
 
```
User
 │
 ▼
AI Health Chatbot
 │
 ├── 🤒 Symptoms Prediction   → "Bro you might have flu, drink water"
 ├── 🧠 Mental Health Analysis → "Stress level: 9/10, please sleep"
 └── ⚖️  BMI / Fitness Advisor  → "Bhai gym jaao"
 │
 ▼
Personalized Health Suggestions
 │
 ▼
📋 Health Report
```
 
---
 
## ✨ Features
 
| Module | What it does |
|--------|-------------|
| 🤒 **Symptom Checker** | Select symptoms → get possible diagnosis + care advice |
| 🧠 **Mental Health Analyzer** | Answer 5 questions → get wellness score + tips |
| ⚖️ **BMI & Fitness Advisor** | Enter height/weight → get custom workout + diet plan |
| 📋 **Health Report** | Combined summary of all your results in one place |
 
---
 
## 🛠️ Tech Stack
 
- **Frontend:** Streamlit
- **Backend Logic:** Python (rule-based ML + scoring)
- **Libraries:** `numpy`, `pandas`, `scikit-learn`, `plotly`
- **Language:** Python 3.12+

## 🚀 How to Run
 
**Step 1 — Clone the repo**
```bash
git clone https://github.com/shubhamnarware67-cmd/ai-health-assistant
cd ai-health-assistant
```
 
**Step 2 — Install dependencies**
```bash
pip install -r requirements.txt
```
 
**Step 3 — Run the app**
```bash
streamlit run health_ai_app.py
```
 
**Step 4 — Open browser**
```
http://localhost:8501
```
 
*That's it. No PhD required.* 🎓
